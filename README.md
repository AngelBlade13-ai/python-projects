# Python Projects

This repository is organized by week so each assignment stays separate and easy to submit.

## Structure

- `python_projects/week_1` through `python_projects/week_6`
- each week contains `project_01`, `project_02`, and `project_03`
- put each completed assignment in its own project folder

## Submission

For the current submission, use `python_projects/week_1`.

## Notes

- Rename project folders if your class requires specific project names.
- Test each project before pushing to GitHub.
