# Week 5

Week 5 assignments are organized into separate project folders.

## Projects

- `project_01`: Musical Time Machine. Scrapes a chart mirror and creates a YouTube playlist, with OAuth setup notes and YouTube Data API fallback handling.
- `project_02`: Practice Price Alert. Uses the static App Brewery Instant Pot practice page, parses the title and price, and sends a personal test email when the price is below the target.
- `project_03`: Selenium Browser Automation. Practices Selenium setup, element location, page interaction, and a Cookie Clicker automation exercise.

Each project includes its own `README.md` and `requirements.txt`.
